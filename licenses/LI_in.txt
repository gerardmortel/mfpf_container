"Rasio n/m" berarti bahwa Pemegang Lisensi menerima beberapa jumlah ('n') kepemilikan untuk Program Pendukung untuk setiap jumlah ('m') kepemilikan yang ditetapkan dari Program Utama secara keseluruhan. Kecuali apabila ditetapkan lain, jumlah kepemilikan untuk Program Pendukung dibulatkan hingga kelipatan 'n'. Sebagai contoh, apabila Program mencakup 100 PVU untuk sebuah Program Pendukung untuk setiap 500 PVU yang didapatkan dari Program Utama dan Pemegang Lisensi memperoleh 1.200 PVU Program, Pemegang Lisensi dapat memasang Program Pendukung dan memiliki inti prosesor yang tersedia untuknya atau dikelola olehnya hingga 300 PVU. PVU-PVU tersebut tidak perlu diperhitungkan sebagai bagian dari total persyaratan PVU untuk pemasangan Program oleh Penerima Lisensi untuk kepentingan pemasangan Program Pendukung (meskipun PVU-PVU tersebut mungkin perlu diperhitungkan karena alasan lainnya, seperti inti prosesor yang disediakan untuk Program Utama juga).

"Server Virtual" adalah unit ukuran yang olehnya Program Pendukung dapat diberi Lisensi. Server adalah komputer dalam bentuk fisik yang terdiri atas unit pemrosesan, memori, dan kemampuan input/output dan yang menjalankan prosedur, perintah atau aplikasi yang diminta untuk seorang atau beberapa orang pengguna atau untuk sebuah atau beberapa perangkat klien. Apabila rak, blade enclosure, atau peralatan serupa lainnya sedang dijalankan, masing-masing alat dalam bentuk fisik yang dapat dipisahkan (sebagai contoh, blade atau alat yang tersusun dalam rak) yang memiliki komponen-komponen yang diperlukan dianggap sebagai server yang terpisah. Server virtual adalah komputer virtual yang diciptakan dengan melakukan partisi atas sumber-sumber yang tersedia untuk server fisik atau server fisik yang tidak dipartisi.

"Penggunaan berdasarkan Program Utama" berarti bahwa Program Pendukung diberikan secara eksklusif untuk digunakan berdasarkan Program Utama. Tidak ada satu Pemegang Lisensi atau aplikasi, program, atau alat apa pun yang berhak menggunakan atau mengakses secara langsung layanan Program Pendukung, kecuali Pemegang Lisensi dapat mengakses Program Pendukung untuk melaksanakan fungsi administratif untuk Program Pendukung seperti pencadangan, pemulihan, dan konfigurasi yang sah.

"Panggilan API Bulanan" adalah unit ukuran yang olehnya Program Pendukung dapat diberi lisensi. Suatu Panggilan API adalah pemanggilan Program melalui antarmuka programabel. Kepemilikan yang memadai harus diperoleh untuk mencakup jumlah maksimum Panggilan API dalam bulan kalender apa pun.
